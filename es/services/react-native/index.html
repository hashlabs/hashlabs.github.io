REACT NATIVE

¿Qué es React Native?

React Native es un framework móvil que te permite crear aplicaciones móviles nativas, usando JavaScript y React para que tanto Android como iOS compartan la misma base de código. Utiliza la misma sintaxis que React, así que si su equipo ya usa React no van a notar el cambio.

¿Por qué React Native?

React Native ofrece una forma flexible de crear aplicaciones nativas con un ciclo de desarrollo rápido. Ya que está escrito en JavaScript, comparte el mismo ciclo de "escritura y recarga" que los desarrolladores web están acostumbrados, reduciendo mucho el tiempo que tarda en desarrollar nuevas características.

¿Cuándo usamos React Native?

  • Lo usamos para todas las aplicaciones móviles que desarrollamos.
