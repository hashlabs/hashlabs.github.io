REACT

¿Qué es React?

React es un framework para applicaciones front-end creado por Facebook y usado en muchos de sus productos, como Instagram y facebook.com. React te permite crear componentes pequeños que combinándolos construyen una aplicación más compleja de manera fácil y ordenada.

¿Por qué React?

React nos permiten crear interfaces web (y en móviles con React Native) rápidamente, y cuando se usan con otras herramientas como Redux se convierte en un framework de front-end completo. La mayor parte de los proyectos que realizamos utilizan React, ofreciendo experiencias ricas que funcionan incluso en dispositivos de gama baja.

¿Cuándo usamos React?

  • En cada aplicación que necesita una interfaz de usuario.
