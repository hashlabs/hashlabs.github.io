What is React?

React is the number one framework to create web application today, it gives us the tools to make declarative and composable interface components that can combined to create complex interfaces.

React comes from the experience of building web applications at Facebook where it was designed with performance and developer happines on mind.

Why we use React?

React allow us to create web (and mobile with React Native) interfaces very quickly, and when used with other tools like Redux it becomes a full front-end framework. Most of our project use React today, delivering rich experiences that work even on the low-end devices.

When we use React?

  • Pretty much in every application that needs UI.
